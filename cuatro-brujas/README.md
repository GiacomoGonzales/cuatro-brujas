# ✨ Cuatro Brujas

Una aplicación web mágica y moderna desarrollada con React, Vite y TailwindCSS que conecta el mundo místico con la tecnología moderna.

## 🎯 Características

- **React 18** con Vite para desarrollo rápido
- **TailwindCSS** con configuración personalizada y colores místicos
- **Framer Motion** para animaciones fluidas y mágicas
- **React TSParticles** para efectos de fondo animados
- **React Router DOM** para navegación entre brujas
- **Google Fonts** (Playfair Display) para tipografía elegante

## 🧙‍♀️ Las Cuatro Brujas

### 1. Elvira - Maestra del Tarot 🔮
- **Ruta**: `/tarot`
- **Especialidad**: Lectura de cartas del tarot
- **Futuro**: Interpretación personalizada con IA

### 2. Morgana - Intérprete de Estrellas ⭐
- **Ruta**: `/carta-astral`
- **Especialidad**: Carta astral y astrología
- **Futuro**: Mapa astral personalizado

### 3. Selene - Guardiana de los Números 🔢
- **Ruta**: `/numerologia`
- **Especialidad**: Numerología y números del destino
- **Futuro**: Cálculos numerológicos avanzados

### 4. Luna - Vidente Lunar 🌙
- **Ruta**: `/horoscopo`
- **Especialidad**: Horóscopo diario y predicciones
- **Futuro**: Predicciones personalizadas

## 🚀 Instalación y Desarrollo

```bash
# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Previsualizar build de producción
npm run preview
```


## 🔮 Próximas Características

- **Integración con IA**: Cada bruja tendrá su propia personalidad y respuestas generadas por IA
